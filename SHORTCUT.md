# Shortcuts — Nixvim

## File tree (nvim-tree)

| Key     | Action                          |
|---------|---------------------------------|
| `<Tab>` | Toggle tree on/off              |
| `<CR>`  | Open file / folder              |
| `<Space>` | Open file / folder            |
| `o`     | Open file / folder              |
| `/`     | Filter tree (live filter)       |
| `E`     | Expand all folders              |
| `W`     | Collapse all folders            |
| `-`     | Go up one directory             |
| `q`     | Close tree                      |
| `H`     | Toggle dotfiles                 |
| `r`     | Rename file/dir                 |
| `d`     | Delete file/dir                 |
| `y`     | Copy filename                   |
| `gy`    | Copy absolute path              |
| `s`     | Run system command on file      |
| `f`     | Start live filter               |
| `F`     | Clear live filter               |

## Fuzzy finder (Telescope)

| Key          | Action                        |
|--------------|-------------------------------|
| `<leader>ff` | Find files (by name)          |
| `<leader>fg` | Live grep (search content)    |
| `<leader>fb` | Browse open buffers           |
| `<leader>fh` | Help tags                     |

## General Vim

| Key            | Action                  |
|----------------|-------------------------|
| `:`            | Command mode            |
| `/<pattern>`   | Search forward          |
| `?<pattern>`   | Search backward         |
| `<C-n>`        | Toggle tree (internal)  |
