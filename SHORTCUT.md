# Shortcuts

## File tree (nvim-tree)

| Key         | Action                |
|-------------|-----------------------|
| `<Tab>`     | Toggle tree on/off    |
| `<leader>e` | Toggle tree on/off    |
| `<CR>`      | Open file / folder    |
| `<Space>`   | Open file / folder    |
| `/`         | Filter tree (live)    |

## AvanteChat

| Key         | Action                    |
|-------------|---------------------------|
| `<leader>d` | Toggle Avante chat        |
